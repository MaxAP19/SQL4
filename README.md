# SQL4
DML language in SQL

## Описание:
Проект посвящён работе с изменением данных в SQL с использованием DML-операций и объединений таблиц. В рамках заданий выполнялся анализ заказов и посещений, а также добавление, обновление и удаление данных в реляционной базе.

## Этапы выполнения проекта:
- Анализ данных с использованием JOIN и операций над множествами для сравнения посещений и заказов.
- Поиск «забытых» меню, пицц и пиццерий без заказов.
- Добавление новых данных с помощью INSERT, включая insert-select и динамический расчёт идентификаторов.
- Обновление данных с использованием UPDATE (изменение цен).
- Удаление данных с помощью DELETE и откат изменений для сохранения целостности базы.

## Освоенные компетенции:
- Практическое применение DML-операций: INSERT, UPDATE, DELETE.
- Работа с данными в нескольких связанных таблицах с использованием JOIN.
- Использование операций над множествами (UNION, EXCEPT, INTERSECT) для аналитических задач.
- Навыки безопасного изменения данных и понимание последствий DML-операций.
- Понимание связи SQL-операций изменения данных с принципами реляционной теории.
